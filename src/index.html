Menggambar Garis

  1. Klik tombol Line
  2. Pilih dua titik pada canvas

Menggambar Persegi

  1. Klik tombol Rectangle
  2. Pilih satu titik pada canvas
  3. Atur panjang dan lebar dari persegi dengan menggerakkan mouse
  4. Jika sudah sesuai klik pada canvas

Menggambar Polygon

  1. Klik tombol Polygon
  2. Pilih titik-titik polygon pada canvas
  3. Jika poligon sudah selesai digambar klik kanan pada canvas atau bisa mengklik tombol finalize polygon

Menyimpan Model

  1. Klik tombol Save untuk menyimpan model yang telah digambar

Memuat Model

  1. Klik tombol Pilih File
  2. Pilih model yang ingin dimuat
  3. Setelah model terpilih klik tombol Load

Memilih Warna Shape

  1. Klik tombol Color Picker yang berada di sebelah tombol Bucket
  2. Pilih warna yang diinginkan
  3. Gambar shape yang diinginkan

Mengubah Warna Shape

  1. Klik tombol Bucket
  2. Klik salah satu vertex pada shape yang warnanya ingin diubah
  3. Klik semua vertex pada pada shape tersebut agar warna shape tersebut berubah

Menggeser Titik Kontrol/Simpul dengan Mouse

  1. Klik tombol Select
  2. Klik salah satu vertex pada shape yang simpulnya ingin digeser
  3. Atur pergeseran simpul dengan menggerakkan mouse
  4. Jika telah sesuai klik pada canvas
